Python

NBA Positional Impact Analysis

Built an end-to-end data pipeline analyzing 1,913 NBA player-seasons of on/off net-rating data to quantify which position most influences team outcomes. Applied formal statistical testing (ANOVA, Kruskal-Wallis, Levene's test) and cross-validated against regularized RAPM, uncovering a measurement artifact in raw on/off splits that only surfaced after applying the cross-check. When the primary API failed mid-project, I re-architected the entire data layer around an alternative dataset, validating it against the same conceptual measure to preserve methodological rigor over expedience.

R

NBA Salary Econometric Study

Investigated wage determinants for NBA players using the Wooldridge nbasal dataset, running OLS regressions on log-wage against player position with controls for age, experience, points, games played, and demographics. Tested for omitted variable bias across multiple specifications via auxiliary regressions, then implemented 2SLS instrumental-variable regression using draft position as an instrument. Diagnosed instrument validity using Wu-Hausman and weak-instrument tests, correctly identifying that a 466% IV coefficient signaled exclusion-restriction violations rather than a true effect.
